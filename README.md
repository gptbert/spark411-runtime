# spark411-runtime

Spark 4.1.1 + Java 21 + Python 3.12 runtime bundle for mixed CentOS 6/7/8 YARN clusters.

This repository includes:
- Docker build context
- self-contained runtime scripts for glibc side-loading
- smoke tests
- submit templates for Spark 4.1.1 on YARN
- GitHub Actions workflow to build, test, and publish a Docker image to Docker Hub

See the full README committed in later steps if this initialization commit is minimal.
